### 📝 Introdução do Projeto
Este CheckPoint (CP1) tem como objetivo a criação de uma **Política de Segurança da Informação (PSI)** para uma empresa fictícia. A PSI é um documento essencial em qualquer organização, servindo como base para a gestão de riscos e conformidade.

> "A segurança da informação não é apenas um produto, mas um processo contínuo de gestão de pessoas, processos e tecnologias."

---

### 🎯 Objetivos da PSI Desenvolvida
- [x] Estabelecer diretrizes claras de segurança.
- [x] Definir responsabilidades para todos os níveis da organização.
- [x] Garantir o comprometimento da Alta Direção.
- [x] Implementar controles de acesso e proteção de dados.
- [x] Estabelecer sanções para o descumprimento das normas.
